# JP Tech Academy

Modern Academy Website with Next.js, Tailwind CSS, Node.js, and MySQL.

## Features
- Public Website (Home, About, Courses, Batches, Contact)
- Student System (Register, Login, Dashboard)
- Admin System (Manage Courses, Batches, Students)
- Mobile Responsive Design

## Tech Stack
- Frontend: Next.js + Tailwind CSS
- Backend: Node.js
- Database: MySQL
- Hosting: Vercel + Railway

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
